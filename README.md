# MagicMirror² on Raspberry Pi Zero 2 W

This repository contains the setup and configuration for a MagicMirror² instance running on a **Raspberry Pi Zero 2 W**.

Given the hardware constraints of the Zero 2 W (512MB RAM), this project is optimized to run on **Raspberry Pi OS Lite (32-bit)** to ensure maximum performance and stability.

---

## 💻 Environment Specifications

- **Hardware:** Raspberry Pi Zero 2 W
- **OS:** Raspberry Pi OS Lite (32-bit)
- **Node.js Version:** 18.x (Recommended for stability)
- **Mode:** Server-only with a lightweight X-server for display.

make script runnable execute this command

chmod +x setup_mirror.sh
